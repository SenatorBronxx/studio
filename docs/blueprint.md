# **App Name**: Eritas Gateway

## Core Features:

- Sign-In: Allow existing users to sign in using email and password.
- Sign-Up: Allow new users to create an account using email and password.
- Social Sign-Up: Allow users to sign up/in with their Google or Apple accounts.
- Logo Display: Display the 'Eritas' logo prominently on the sign-in/sign-up page.
- Animation Tool: AI-powered module to add personalized sticker and other animated customizations to enhance UX based on preferences.

## Style Guidelines:

- Primary color: Forest green (#228B22), taken from the logo, evokes trust and natural associations.
- Background color: Off-white (#FAF9F6), is calm and complementary.
- Accent color: Lime green (#32CD32), a lively, verdant shade of spring that stands out from the primary, calling attention to key UI elements.
- Body and headline font: 'Inter' sans-serif; clean, modern, objective, neutral, readable, and space-saving.
- Use transportation-themed icons such as buses and location pins for a clear visual association with the app's purpose.
- Subtle animated stickers that represent travel (e.g., a bus moving across the screen, a spinning wheel). Keep animations smooth and non-distracting.
- Implement a clean, minimalist layout with a focus on user-friendly input fields and clear call-to-action buttons.
